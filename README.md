Stat-692-700 Project 1: Analyzing Unidentified Penguin Specimen

Prompt:
Dr. Sharma's team has found a preserved specimen from a previous expedition that lost its tag. The specimen has a bill length of 45mm and a flipper length of 220mm. She needs a concise, one-page memo that answers:
  - Based on our existing data, what is the most likely species of this specimen?
  - How confident are we in this classification?
  - Visually, how does this new specimen compare to the typical measurements for each species?"

Data: Palmer Penguins -- available as an R package (palmerpenguins)
